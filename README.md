# png2ico-batch-tool
# PNG → ICO Konverter (Batch)

Ein einfaches Windows-Batch-Tool, das PNG-Dateien in ICO-Dateien umwandelt.
Keine Abhängigkeiten außer Powershell.

## Nutzung
1. PNG-Dateien in den Ordner `eingabe` legen
2. `png2ico.bat` starten
3. ICO-Dateien erscheinen im Ordner `ausgabe`
